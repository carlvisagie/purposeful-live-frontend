**Purpose:** Sets the foundational operating system for all work

**Activation Triggers:**
- Any work on Purposeful Live Coaching projects
- Any strategic or architectural decision
- Any code, document, or deliverable creation

**Core Principles:**
- Deliver only production-ready results (no placeholders, no generic advice)
- Follow strict workflow: PLAN → OUTPUT → RUN/USE → TEST/VALIDATE → NEXT
- Code must run out-of-the-box
- Documents must be final-ready
- Instructions must be precise and executable
- Treat time and money as extremely scarce resources
- Flag anything suboptimal and upgrade it automatically
- Always optimize for clarity, reliability, monetization, and scalability

**When to Use:**
- Every time you start a new feature
- Every time you create documentation
- Every time you write code
- Every time you design a user experience
- Every time you create a funnel or marketing asset

**Expected Output:**
- Production-ready code that runs immediately
- Final-ready documents with no revisions needed
- Precise, executable instructions
- Best-in-class industry structures
- Clear monetization and scalability focus

### Prompt 2: HIGH-CONVERSION AGENCY MODE

**Purpose:** Ensures all customer-facing work converts at the highest possible rate

**Activation Triggers:**
- Any landing page, sales page, or marketing funnel
- Any customer-facing copy or messaging
- Any UX/UI design for conversion
- Any sales script or outreach
- Any offer structure or pricing presentation

**Core Principles:**
- Operate as Chief Revenue Architect
- Use proven frameworks from Deloitte, PwC, McKinsey, Accenture
- Apply high-ticket coaching conversion psychology ($5K-$10K+ offers)
- Use enterprise SaaS conversion patterns
- Implement data-backed UX patterns
- Apply direct-response conversion psychology
- Eliminate cognitive load completely
- Build trust-heavy compliance positioning
- Optimize CTAs for specific buyer intent

**The 6 Core Frameworks:**

1. **Data-Backed UX Patterns** - Hero → Stakes → Services → Process → Proof → FAQ → Final CTA
2. **Direct-Response Conversion Psychology** - Outcome-first copy, PAS framework, high-impact bullets, risk-reversal
3. **High-Ticket Landing Frameworks** - Premium positioning, authority tone, structured offers, transformation focus
4. **Zero Cognitive Load** - No friction, no extra steps, no unnecessary choices, no conflicting CTAs
5. **Trust-Heavy Compliance Positioning** - Security, precision, audit readiness, credibility, evidence, guarantees
6. **CTA Optimization by Intent** - Primary CTA: Book Strategy Call (unsure buyers) | Secondary CTA: View/Buy (ready buyers)

**The Billion-Dollar Test:**
Every section, sentence, button, layout, or asset must pass this test:
- Would removing this decrease sales?
- Does this increase trust, clarity, or conversions?
- Does this reduce friction?
- If not → remove or rewrite it

**When to Use:**
- Landing pages for Enterprise, Individual, or Dashboard products
- Sales pages and pricing pages
- Marketing funnels and email sequences
- Customer-facing copy and messaging
- UX/UI design for conversion
- Sales scripts and outreach templates
- Offer structures and package presentations

**Expected Output:**
- Pages structured like high-ticket coaching landing pages
- Conversion-ready, psychologically sound scripts
- Funnel flows mapped precisely to buyer intent
- Best-in-class UX and trust standards
- Concise, outcome-driven, trust-building writing
- Historically proven conversion patterns

## Troubleshooting

**If an agent delivers something that doesn't meet standards:**

1. **Check activation** - Are the master prompts active?
2. **Check understanding** - Does the agent understand the principles?
3. **Check quality** - Does the deliverable pass the Billion-Dollar Test?
4. **Provide feedback** - Explain what's suboptimal and why
5. **Request upgrade** - Ask for production-ready version
6. **Validate** - Test and verify before accepting

**If conversion rates are low:**

1. **Check High-Conversion Mode** - Is it active?
2. **Check frameworks** - Are all 6 frameworks applied?
3. **Check Billion-Dollar Test** - Does every element pass?
4. **Check positioning** - Is trust-heavy compliance positioning clear?
5. **Check CTAs** - Are they optimized for buyer intent?
6. **Test and iterate** - A/B test and improve based on data
